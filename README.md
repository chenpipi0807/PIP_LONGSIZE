## PIP 长边调整节点 PIP 长边调整节点

#### 简介 简介
PIP 长边调整节点是一个用于 ComfyUI 的自定义节点，
主要作用是按照最长边等比例调整图像大小；
同时它也支持调整环节的图像压缩

## 功能
- 调整图像大小，保持原始宽高比 调整图像大小，保持原始宽高比
- 支持批量处理图像 支持批量处理图像
- 提供三种压缩选项：无损输出、中档压缩、最小文件 提供三种压缩选项：无损输出、中档压缩、最小文件
- 默认将最长边调整为 996 像素 默认将最长边调整为 996 像素
